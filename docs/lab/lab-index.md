# Lab — poligon eksperymentów

**Cel:** bezpieczne miejsce na **prompty, prototypy i pomysły**, zanim trafią do `docs/core/` lub kodu.

## Zasady

1. **Lab ≠ produkcja** — nic z Lab nie jest SSoT, dopóki nie przejdzie review i nie trafi do CORE / playbooków.
2. **Jeden eksperyment = jeden wpis** — krótki opis, data, wynik (działa / odrzucone / do dalszej pracy).
3. **Prompty** — trzymaj wersje z datą; najlepsze przenieś do [Prompt formulas](/reference/prompt-formulas).

## Jak promować do core

1. Eksperyment daje powtarzalny wynik (2+ udane sesje).
2. Aktualizujesz odpowiedni dokument w `docs/core/` lub `docs/playbooks/`.
3. Wpisujesz źródło w [System evolution](/system-evolution).

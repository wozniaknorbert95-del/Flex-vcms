# 🛑 W przypadku Chaosu - Procedura IF-LOST

Jeżeli czujesz paraliż decyzyjny (Finish Line Resistance), overthinking, lub system wymknął się spod kontroli, zatrzymaj się natychmiast i przejdź przez tę twardą ścieżkę.

## 1. Reguła Brzydkiego Końca (Ship it ugly)
Publikujesz wersję wydania, która wywołuje lekki dyskomfort. Zabrania się dopracowywania systemu, dopóki nie zostanie on zweryfikowany przez warunki bojowe. 

## 2. Zamknięcie jako Osobny Task
- Wyeliminuj rozproszenia. Ustaw timer na **90 minut**. 
- Dokończ kod, **nawet jeśli to w tym momencie sprawia ból psychiczny**.

## 3. Eskalacja do KODA
- Wróć do modułu Chat Bota **[KODA](/koda)**.
- Wyślij komendę: `Zatrzymałem się. Rozbij to na jeden najprostszy krok i wymuś jego wykonanie.`
- Pamiętaj: kod to eksperyment. Nie zadawaj pytań "czy idealnie?". Sprawdzaj "co się stanie jeśli uruchomię?".

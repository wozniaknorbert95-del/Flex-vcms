# System Flagowania
Każdy projekt w tytule powinien określać swój stan życia:
- `[STABLE]` - kod gotowy na produkcję.
- `[WORKING]` - kod wdrożeniowy w fazie iteracji.
- `[EXPERIMENTAL]` - prace w katalogu Lab, niewpływające na rdzeń.
- `[ARCHIVED]` - zamknięta sprawa / cmentarzysko kodu.

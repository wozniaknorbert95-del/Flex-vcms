# System Evolution
> [!IMPORTANT]
> PROMOTION FROM EXPERIMENTAL TO STABLE

- **[EXPERIMENTAL]**: New logic, unverified files, or raw logs.
- **[PENDING]**: Files undergoing 48h isolation.
- **[STABLE]**: Verified, peer-reviewed, and mission-critical components.

### Quarantine Process
1. Move experimental files to lab/.
2. Conduct **Weekly Review**.
3. If no regression is found, promote to target directory and update YAML status.
